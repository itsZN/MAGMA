MAGMA works by adjusting your code at the IR level, creating and injecting common vulnerabilities and bug classes. With several compile options, it allows for flexable bug injection, letting you customize how vulnerable your program will become. If you want to test your binary analysis platform on a non-trivial project, but would rather not go though the tedious process of adding vulnerabilities yourself, look no further.
